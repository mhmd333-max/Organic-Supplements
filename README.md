🌿 Organic Supplements & Fitness Store

A modern React web application for showcasing organic fitness supplements, vitamins, and health products with a clean and responsive UI.

🚀 Features
🏋️ Trending fitness supplements page
🧬 Organic whey, vitamins, and health products
🛒 Product cards with images, price, and add-to-cart button
📱 Responsive design (mobile + desktop)
⚛️ Built with React (Vite setup)
🎨 Clean and modern UI design
🛠️ Tech Stack
React.js
JavaScript (ES6+)
HTML5
CSS3
Vite
